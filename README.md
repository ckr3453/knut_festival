## 📒 개요
   * 2019년도 국원대동제 "별 달은 밤"을 소개합니다.
   * 축제 시간표, 교내외 부스 소개, 교통정보 등 축제 관련 정보를 제공합니다.
   * 술 친구, 분실물 게시판 등을 자체적으로 서비스합니다.
